Protocol

Experiment structure

The tracker follows your defined schedule exactly: weeks 0-2 baseline, 3-10 intervention, and 11-12 washout. That produces week labels 0 through 12, which is 13 labeled weeks in the app.

  1. Keep medication timing, caffeine timing, creatine, exercise cadence, and sleep schedule as stable as possible.
  2. Record prior night's sleep during the morning check-in, then log energy, brain fog, recovery, and joint stiffness across the day.
  3. Run cognitive tests three times per week using the same tools, same device, and roughly the same time window.
  4. Use notes and anomaly tags whenever illness, travel, poor sleep, or schedule disruption could confound a result.
  5. Interpret the intervention only after comparing against baseline and then checking for regression in washout.

Testing cadence

Recommended measurement rules

Reaction time

Use Human Benchmark or another fixed reaction test. Run three trials on test days and store the average milliseconds.

Working memory

Use digit span as the primary memory metric. PsyToolkit's digit span task is a good fit because it returns a clear span score you can enter directly into the tracker.

For this app, the recommended flow is to open the PsyToolkit digit span test in a new tab, finish the task, and enter the resulting digit span score manually. That is more reliable than trying to embed the test directly into this static dashboard.

Suggested schedule

Monday, Wednesday, and Friday are the default recommended test days in the app. You can still log cognitive data on any day.

Lion's mane schedule

The app prefills capsules by protocol week: 0 for baseline, 2 daily for weeks 3-4, 4 daily for weeks 5-10 with a split dose, and 0 again for washout.

Backup workflow

Export data regularly and commit the snapshot into your repo if you want a versioned record without giving up local-only daily use.
